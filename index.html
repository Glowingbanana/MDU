Connect Firebase in 5 steps

Firebase lets everyone see changes in real time. It's free for small teams.

  1. Go to console.firebase.google.com and click Create a project
  2. Give it a name (e.g. mdu-pmo), disable Google Analytics, click Create project
  3. Click Realtime Database in the left sidebar → Create Database → choose any location → Start in test mode → Enable
  4. Click the ⚙ gear icon → Project settings → scroll to Your apps → click </> (Web) → register app → copy the firebaseConfig object
  5. Open this HTML file in a text editor, find const firebaseConfig = { near the top, and paste your values in

After saving and re-uploading to GitHub, all changes will sync in real time for everyone with the link.
